GitHub – цікава програма для людей, які бажають можливостей створення власних проектів. Спочатку для мене він був важкий і не зрозумілий, що уповільнювало вивчення його можливостей. Згодом після певного часу використання звикаєш і починаєш розуміти. Для тих кому це в новинку, таких як я, це був захоплюючий досвід.
Я навчився використовувати github для швидкої співпраці з другом, створення власного проекту. Зокрема, це чудове місце для знайомсмтв з новими людьми та обговорювання тем, що вас цікавлять з однодумцями.
